# 🏥✨ Hospital Patient Registry System

A modern Hospital Patient Registry System built using **Node.js, Express.js, HTML, and CSS**. This application helps hospitals manage patient records efficiently by allowing patient registration, data storage, and record viewing through a simple and user-friendly interface.

---

## 🌟 Features

🩺 Register New Patients

📋 Store Patient Records Securely

👨‍⚕️ View All Registered Patients

📅 Record Date of Admission

💊 Store Illness Details

📂 File-Based Data Storage

🎨 Clean & Responsive User Interface

⚡ Fast and Easy Record Management

---

## 🛠️ Technologies Used

🌐 HTML5

🎨 CSS3

⚙️ JavaScript

🟢 Node.js

🚀 Express.js

📂 File System (fs Module)

```bash

```

```bash
node app2.js
```

### 3️⃣ Open in Browser

