
# Hospital Patient Registry

~ Hospital Patient Registry using Node.js, Express.js, HTML and CSS with patient registration and record viewing functionality.

## Features

- Register new patients
- Store patient details in a text file
- View all registered patients
- Clean and responsive user interface using CSS
- Patient information includes:
  - Patient Name
  - Date of Admission
  - Illness Name

## Technologies Used

- HTML
- CSS
- Node.js
- Express.js
- File System (fs) Module
